# ideapadconfigtool
Allow to changes mode of battery, Fn Lock and USB charging in IdeaPad laptops with ideapad_lenovo active module of Linux Kernel.

**How to build.**

Run:

```
meson build  --prefix=/usr
cd build
ninja
ninja install
```


Search in Applications menu: IdeadPad Config Tool.
